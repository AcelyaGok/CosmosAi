# CosmosAi
İşte 350 karaktere uygun kısaltılmış hali:  **Cosmos AI**, Python ile geliştirilen ve Telegram’a entegre çalışan bir astroloji chatbotudur. Kullanıcılara kişiselleştirilmiş burç yorumları, günlük bildirimler ve doğum haritası analizleri sunar. NLP ve yapay zekâ desteğiyle akıcı, kullanıcı dostu bir deneyim sağlar.
